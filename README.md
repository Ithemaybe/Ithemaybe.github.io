# 🌍 Флаги Мира — Угадай Страну

Браузерная игра о странах мира с двумя режимами. Никакого бэкенда — чистый HTML, CSS и JavaScript, задеплоено через GitHub Pages.

**[▶ Играть →](https://ithemaybe.github.io/game/)**

---

## Режимы игры

### 🏳️ V1 — Угадай по флагу
Показывается флаг страны и 4 варианта ответа. Нужно выбрать правильный. **60 секунд** — сколько успеешь?

- Управление мышью, тапом или с клавиатуры (1–4 / стрелки)
- Счётчик правильных и ошибочных ответов в реальном времени
- Круговой таймер
- Экран результатов с процентом точности

### 🗺️ V2 — Назови страны
Чистое поле ввода, никаких подсказок. Вспомни как можно больше стран мира.

#### ⏱️ Выбор времени
На старте выбери продолжительность раунда:
- **1 минута** — спринт на скорость
- **5 минут** — сбалансированный режим
- **10 минут** — марафон памяти (по умолчанию)

#### 🎙️ Режим ввода
Выбери способ, которым будешь называть страны:

- **⌨️ Письменный** — вводи названия с клавиатуры и нажимай Enter (как раньше)
- **🎙️ Голосовой** — произноси названия вслух, игра распознаёт речь на русском языке.
  Требуется разрешение браузера на доступ к микрофону.
  Поддерживается в Chrome, Edge и Safari (Web Speech API).

#### Прочие особенности
- Принимаются названия на русском языке (включая псевдонимы: «Голландия» → Нидерланды, «Бирма» → Мьянма и т.д.)
- Повторы и несуществующие страны не засчитываются
- Принятые страны отображаются тегами в реальном времени
- Прогресс-бар (всего стран в мире: 195)
- Личный рекорд сохраняется в localStorage

В обоих режимах результат можно скопировать как ссылку и кинуть вызов другу.

## Структура проекта

```
game/
├── index.html        # V1 — угадай страну по флагу
├── game.js           # Логика V1
├── v2.html           # V2 — назови страны по памяти
├── v2.js             # Логика V2 (выбор времени + голосовой ввод)
├── style.css         # Стили для V1
├── style-v2.css      # Стили для V2
├── swimming-cat.js   # Плавающий котик
├── result.html       # Страница результата (шэринг)
└── 404.html          # Кастомная страница ошибки
```

## Технологии

- Чистый **HTML / CSS / JavaScript** — без фреймворков и зависимостей
- Флаги — публичный CDN [flagcdn.com](https://flagcdn.com)
- Шрифты — Unbounded + Inter (Google Fonts)
- Голосовое распознавание — [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API)

## Запуск локально

```bash
git clone https://github.com/Ithemaybe/game.git
cd game
# Открыть index.html в браузере или запустить любой локальный сервер:
npx serve .
```

> ⚠️ Голосовой режим требует **HTTPS** или `localhost`. При открытии файла напрямую через `file://` Web Speech API недоступен.
